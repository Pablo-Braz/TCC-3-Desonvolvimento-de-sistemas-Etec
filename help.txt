# Guia Rápido do Projeto

## 1) Instalar dependências
- Backend (PHP):
  - composer install
- Frontend (Node):
  - npm install
  - npm run build (ou npm run dev para ambiente de desenvolvimento)

## 2) Preparar a aplicação
- Gerar chave da aplicação:
  - php artisan key:generate
- Rodar servidor embutido do Laravel:
  - php artisan serve

## 3) Banco de Dados
- Migrar estrutura:
  - php artisan migrate
- Popular dados básicos com o seeder (AutoPopulateSeeder):
  - php artisan db:seed --class=Database\Seeders\AutoPopulateSeeder
- Resetar e popular tudo de uma vez (opcional):
  - php artisan migrate:fresh --seed
  --possivel local para hospedagem -- AlwaysData 

Observação: O seeder cria 1 usuário admin na tabela `usuario`, 1 comércio, 10 categorias, 30 clientes, 100 produtos e seus estoques. Vendas e contas fiadas não são criadas automaticamente.

---

## 4) Observações úteis
- Caminho das classes do Bootstrap: `node_modules/bootstrap/dist/css/bootstrap.css`
- Comandos frontend: `npm run dev`
- Animações: manter escalas adequadas conforme a UI
- Tema VS Code sugerido: Synthesis

## 5) Dica: Composer mais rápido (Windows/Laragon)
1. Abra o arquivo:
   C:\laragon\bin\php\php-8.3.4-Win32-vs16-x64\php.ini
2. Ative a extensão ZIP removendo o `;` do início da linha:
   ;extension=zip => extension=zip
