# TCC-3-Desonvolvimento-de-sistemas-Etec
Gestão de Comércio - Sistema Completo Este repositório contém um sistema completo para gestão de comércios, desenvolvido com Laravel (backend) e ferramentas modernas de frontend (Node.js, Vite). O projeto inclui funcionalidades como gerenciamento de usuários, produtos, categorias, estoques e vendas.
